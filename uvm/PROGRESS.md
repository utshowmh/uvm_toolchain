# UVM

</br>

**FEATURES:**

  </br>

- **Can Load Program from File**

  ---

  Can load a file full of instruntions [an instruction is an operation with operand if it has any] (separated by new line) and has ability to execute them.
  
  </br>

- **Can Execute**
  
  ---

  - push
  - pop
  - duplicate
  - swap
  - jump
  - jumpif
  - eql
  - plus
  - minus
  - multiplication
  - division
  - out
  - dump
  - halt
  - .label: for defining label
  - and ; for commenting
